# Smart Water Quality and Usage Recommendation System

A Flask-based web application that analyzes water quality based on pH and TDS values, estimates mineral composition, and recommends suitable usage (drinking, domestic, or industrial).

### Features
- User Login and Signup
- Water Analysis and Quality Classification
- Mineral Composition Estimation
- Historical Data Visualization using Chart.js
- CSV-based Local Storage

Developed by Aman, Aditya, and Sameer.
